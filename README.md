# 📱 Mis Recetas - Web App

Tu app de recetas con despensa inteligente, ahora como Progressive Web App (PWA).

## ✨ Características

- ✅ Gestión completa de recetas (ver, buscar, filtrar, eliminar)
- ✅ Despensa inteligente (inventario de ingredientes)
- ✅ Lista de compras
- ✅ Sistema de coincidencias (% de ingredientes disponibles)
- ✅ Filtros: "Puedo hacer", "Casi puedo hacer"
- ✅ Búsqueda sin acentos
- ✅ Funciona offline
- ✅ Instalable en Home Screen
- ✅ Funciona en iPad + iPhone + cualquier dispositivo

## 🚀 Cómo instalar (GRATIS)

### Paso 1: Crear cuenta en GitHub
1. Ve a https://github.com
2. Crea una cuenta gratuita (si no tienes)
3. Verifica tu email

### Paso 2: Subir los archivos
1. En GitHub, crea un nuevo repositorio llamado "recetas-app"
2. Márcalo como "Public"
3. Sube todos estos archivos:
   - index.html
   - styles.css
   - app.js
   - manifest.json
   - sw.js

### Paso 3: Activar GitHub Pages
1. Ve a Settings (Configuración) del repositorio
2. En el menú izquierdo, busca "Pages"
3. En "Source", selecciona "main" branch
4. Guarda
5. Espera 2-3 minutos

### Paso 4: ¡Listo!
Tu app estará disponible en:
```
https://TU-USUARIO.github.io/recetas-app
```

## 📱 Añadir a Home Screen

### En iPhone/iPad (Safari):
1. Abre la URL en Safari
2. Toca el botón "Compartir" (cuadrado con flecha)
3. Desplázate y toca "Añadir a pantalla de inicio"
4. Dale un nombre y toca "Añadir"
5. ¡Ya aparece como app en tu Home Screen!

### En Android (Chrome):
1. Abre la URL en Chrome
2. Toca el menú (⋮)
3. Toca "Añadir a pantalla de inicio"
4. Confirma

## 🎨 Personalización

Todos los archivos son editables. Para cambiar el diseño:

**Colores** → Edita `styles.css` (líneas 10-20, variables CSS)
**Funcionalidad** → Edita `app.js`
**Estructura** → Edita `index.html`

## 💾 Almacenamiento

Todos los datos se guardan en el navegador (localStorage):
- Las recetas persisten aunque cierres la app
- Funciona sin conexión
- Los datos NO se sincronizan entre dispositivos

## 🔄 Actualizaciones

Para actualizar la app:
1. Edita los archivos en GitHub
2. Los cambios se reflejan automáticamente en 1-2 minutos
3. Los usuarios verán la nueva versión al recargar

## ⚠️ Limitaciones vs App Nativa

- ✅ Todas las funciones principales funcionan
- ⚠️ Cámara funciona pero experiencia inferior
- ❌ No incluye reproductor de música integrado
- ❌ Datos no se sincronizan entre dispositivos

## 🆘 Soporte

Si tienes problemas:
1. Asegúrate de usar HTTPS (GitHub Pages lo hace automático)
2. Borra caché del navegador
3. Prueba en modo incógnito
4. Verifica que todos los archivos estén subidos
